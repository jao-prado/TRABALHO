# Pizzaria Barueri

Sistema web de pizzaria local em Barueri/SP, organizado em tres pastas:

- `frontend`: aplicacao React + Vite.
- `backend`: API Java Spring Boot.
- `sql`: scripts SQL Server.

## Ordem De Desenvolvimento

1. Sprint 0: base e arquitetura.
2. Sprint 1: banco de dados.
3. Sprint 2: autenticacao backend.
4. Sprint 3: CRUD base.
5. Sprint 4: pedidos backend.
6. Sprint 5: frontend base.
7. Sprint 6: cardapio e carrinho.
8. Sprint 7: finalizacao de pedido.
9. Sprint 8: admin e atendente.
10. Sprint 9: revisao geral.
11. Sprint 10: entrega final.

## Dominio Oficial

Usar os nomes abaixo em todas as camadas:

- Usuario
- Cliente
- Atendente
- Admin
- Produto
- Categoria
- Pedido
- ItemPedido
- Endereco

O backend e a fonte de verdade para regras de negocio, seguranca e validacao.
